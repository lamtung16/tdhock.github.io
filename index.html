Machine learning and data visualization consulting services

I am available for hire as a consultant for

  • Programming R packages or Python modules (including C/C++ code). For example I designed and wrote the ggplot interface of the plotly R package.
  • Interactive, animated data visualization. Rapid prototyping using my animint R package. Custom D3 visualizations (for example SegAnnDB).
  • Developing highly accurate predictive models for big data, using machine learning and numerical optimization algorithms. For example my PeakSegJoint R package includes a discrete optimization algorithm in C code, and a convex optimization algorithm in R code.
  • Teaching any of the above subjects. List of previous teaching.

To schedule a free first meeting to estimate my fees for your project, please contact me via email with:

  • a list of times you would be available to meet (Skype, Google Hangout, or face-to-face in Montreal), and
  • a description of your project.
